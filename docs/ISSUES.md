## 이슈 작성 방법

이슈 발생 시 "진행 중 이슈" 테이블에 추가합니다.

```
| 1 | 🟡 중간 | Stage 1 | 이슈 제목 | 2026-xx-xx | 조사 중 |
```

이슈마다 아래 형식으로 상세 내용을 작성합니다.

```markdown
### #1 상세

**현상**:
**우회 방법**:
**원인 후보**:
**현재 상태**:
**향후 조치**:
```

해결되면 해결된 이슈 섹션으로 옮기고 해결 방법을 기록합니다.
